You are designing a board game. The tokens move around the square board with this game.(like Monopoly(Wikipedia))

You need to the function that return the coordinate after the token have moved. The coordinate is a list of row and column. The top-left is [0, 0].

You are given three arguments: the first one is the length of one side of game board (an integer), the second is current position of token (an integer. The cells around the board is numbered clockwise. That starting with 0 on bottom-right.). the third is the number of steps to advance (an integer. positive number is clockwise. negative number is anticlockwise.).
